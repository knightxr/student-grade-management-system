Student Grade Management System

Download the runnable JAR and read the quick-start notes below.

⬇️ Download JAR

How to run

Requires Java 17+ installed. Then double-click the JAR or run:

java -jar StudentGradeManagementSystem-all.jar

If your browser opens the JAR as text, right-click the link and choose Save link as…

Quick start (credentials)

  • Login: username admin, password admin.
  • Create account: when prompted for the administrator password, type admin.

Included data

The database ships with ~300 students across 3 grades.

To test features, you may need to…

  • Add students to courses/classes.
  • Create assignments.
  • Enter marks rewarded for those assinments.
  • Write feedback/comments.

Reports (Microsoft Word required)

To use the Create Report feature, you must have Microsoft Word installed on the computer.
